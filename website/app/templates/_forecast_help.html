# (прогнозное место)
Ранг по ожидаемому счёту E[T] среди команд на этой странице: 1 = самый высокий прогноз.
E[T]
Ожидаемое число взятых вопросов: сумма по пакету вероятностей pq (noisy-OR + lapse + калибровка). Не целое число — это матожидание, не «прогноз округлённого счёта».
{% if n_mc_samples %}
счёт 5–95%
Интервал по {{ n_mc_samples }} симуляциям Монте-Карло: в каждой θ игроков слегка случаются вокруг оценки (шире у новичков), затем по каждому вопросу бросается монетка с вероятностью pq. 5% и 95% — границы распределения итогового счёта.
место 5–95%
Тот же эксперимент, но после каждой симуляции команды ранжируются по счёту (как в таблице турнира). Показывает, на каких местах команда могла бы оказаться при такой неопределённости.

Игроков
Сколько игроков состава попали в нашу базу с известным θ; если после слэша — сколько было в активном составе на турнире.
Факт. счёт / место
Реальный результат того турнира (для сравнения с прогнозом «как бы сыграли сейчас»).
Δ места
Фактическое место минус прогнозное. +N — сыграли лучше, чем предсказывает модель с текущими θ; −N — хуже.
